## DSA Project Source 

제공 되는 데이터는 실제 데이터를 기반으로 학습에 맞게 가공한 데이터이며, 데이터에 명시된 제품명/지역/사람이름 등은 실제와 무관합니다.
DSA에서 제공되는 데이터 및 비즈니스 시나리오, 템플릿 등은 데이터스테이션의 자산입니다. 
### 무단으로 복제,배포 시 법적 책임을 질 수 있습니다.

- 블로그 / Git Hub 등에 제공된 데이터를 직접 올리는 행위
- 다른 공모전, 경진대회 등에 제공된 데이터를 사용하는 행위
- 온라인 채팅 및 커뮤니티에 제공된 데이터를 공유하는 행위
- 수익 및 사업을 목적으로 해당 데이터를 사용하는 행위
- 위의 사항을 포함하여, 제공된 데이터를 무단으로 복제 및 공유하는 기타 모든 행위

다만 데이터 분석 수행 결과, 프로젝트를 수행하며 직접 작성한 PPT (발표자료)등은 포토폴리오에 사용 가능하니 참조 바랍니다.
